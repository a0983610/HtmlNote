Git常用基本指令

download git https://git-scm.com/

指令 說明
git help 指令名稱 查看指令的文件
git init 建立GIT管理機制
git add 加入追蹤
#git add . 可以加入所有此資料夾的檔案
git status 查看上次commit到目前的檔案變更狀態
git commit -m "msg" 建立一次儲存紀錄 必須要輸入資訊
git checkout "branch1" 切換分支
#git checkout -b "branch1" 代表如果沒有該分支則建立並切換
#git checkout master "fileName" 將此檔案改回該分支版本
git branch "branch1" 建立分支
#git branch 察看目前有的分支
#git branch -d "branch1" 刪除分支
如果該分支沒有被merge 需要-D強制刪除
git merge --no--ff "branch1" 將branch整合進目前分支 可以一次合併多個 Branch
--no-ff代表不要將分支上的commit點合併進來
git push 將目前本機狀態上到遠端主機
#在master時 git push即可

在https://github.com/登入帳號 建立Repositories
可以取得web URL. 存到遠端
